A beautiful mobile application designed for sharing notes, tracking progress, and staying connected with loved ones. Built with React Native and Expo, and styled with NativeWind.

## ✨ Features

- **Progress Map**: A visual journey of your notes and milestones.
- **Notes**: Create and view special notes with rich interaction.
- **Chat**: Real-time communication interface for quick replies and reactions.
- **Dark Mode**: Fully supports a sleek, eye-friendly dark theme.
- **Cross-Platform**: Designed to work on iOS, Android, and Web.

## 🚀 Tech Stack

- **Framework**: [Expo](https://expo.dev/) / [React Native](https://reactnative.dev/)
- **Styling**: [NativeWind](https://www.nativewind.dev/) (Tailwind CSS for React Native)
- **Navigation**: [Expo Router](https://expo.github.io/router/docs/)
- **State/Storage**: React Hooks & Async Storage
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🛠️ Project Structure

- `app/`: Contains the main application routes and screens using Expo Router.
- `components/`: Reusable UI components (Buttons, Progress bars, etc.).
- `assets/`: Static assets like images and fonts.
- `utils/`: Helper functions, API calls, and local storage management.
- `styles/`: Global styles and configuration.

## 🏁 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn
- [Expo Go](https://expo.dev/client) app on your mobile device (optional, for testing on real devices)

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm start
```

From the Expo CLI menu, you can:

- Press `i` to open in the iOS simulator.
- Press `a` to open in the Android emulator.
- Press `w` to open in the web browser.
- Scan the QR code with the Expo Go app to run on a physical device.
